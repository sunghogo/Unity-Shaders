# **Unity Shaders**

> WebGL Unity Application showcasing custom shader and lighting methods implementing using Unity Cg/HLSL.

### [Play](https://github.com/sunghogo/Unity-Shaders-WebGL)

### Controls:

| Key    | Controls          |
| ------ | ----------------- |
| W or ↑ | Zoom In           |
| S or ↓ | Zoom Out          |
| A or ← | Rotate Left       |
| D or → | Rotate Right      |
| SPACE  | Cycle Next Shader |
